# Software Squad — Bundled Copy

This is a bundled copy of the Software Squad, shipped inside the BuildPact npm package
to enable offline and zero-config installation.

**Source of truth:** https://github.com/buildpact/buildpact-squads

To get the latest version of this Squad, run:

```bash
npx buildpact squad add software
```

## Agents

- **PM** — Product Manager: specification, epics, user stories
- **Architect** — System design, technical decisions, architecture docs
- **Developer** — Implementation, code review, testing
- **QA** — Quality assurance, test strategy, bug triage
- **Tech Writer** — Documentation, changelogs, user guides
