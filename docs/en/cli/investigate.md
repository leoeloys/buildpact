# buildpact investigate

Research domain, codebase, or technology.

## Usage

```bash
buildpact investigate [options]
```

## Options

| Flag | Description |
|------|------------|
| `--topic` | Focus the research on a specific topic |

## Examples

```bash
# Investigate the current codebase
buildpact investigate

# Research a specific topic
buildpact investigate --topic "authentication best practices"

# Investigate a technology
buildpact investigate --topic "PostgreSQL indexing strategies"
```
