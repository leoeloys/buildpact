# buildpact diff

Track changes between pipeline runs.

## Usage

```bash
buildpact diff
```

## Options

This command has no options.

## Examples

```bash
# Show changes since last pipeline run
buildpact diff
```

Shows what changed since the last spec, plan, or execute run, helping you understand the delta between pipeline stages.
