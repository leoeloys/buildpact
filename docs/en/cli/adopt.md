# buildpact adopt

Onboard an existing project.

## Usage

```bash
buildpact adopt
```

## Options

This command has no specific options.

## Examples

```bash
# Adopt the current directory as a BuildPact project
buildpact adopt
```

Scans your project structure, detects the tech stack, and pre-fills the constitution with sensible defaults based on what it finds.
