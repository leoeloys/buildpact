# buildpact constitution

Create or edit project rules.

## Usage

```bash
buildpact constitution
```

## Options

This command has no options.

## Examples

```bash
# Open the interactive constitution editor
buildpact constitution
```

Launches an interactive editor for managing your project's coding standards, compliance rules, architecture constraints, quality gates, and domain-specific rules.
