# buildpact quality

## Uso

```bash
buildpact quality
```

## Opções

## Exemplos

```bash
# Gerar um relatório de qualidade
buildpact quality
```

Analisa o projeto e produz um relatório de métricas de qualidade inspirado nos padrões ISO 9001.
