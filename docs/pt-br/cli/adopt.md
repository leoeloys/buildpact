# buildpact adopt

Integrar um projeto existente.

## Uso

```bash
buildpact adopt
```

## Opções

Este comando não possui opções específicas.

## Exemplos

```bash
# Integrar o diretório atual como um projeto BuildPact
buildpact adopt
```

Analisa a estrutura do seu projeto, detecta a stack tecnológica e preenche a constituição com padrões sensatos com base no que encontra.
