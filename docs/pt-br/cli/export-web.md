# buildpact export-web

Exportar como bundle web para interfaces web.

## Uso

```bash
buildpact export-web [opções]
```

## Opções

| Flag | Descrição |
|------|-----------|
| `--compress` | Nível de compressão do bundle |

## Exemplos

```bash
# Exportar um bundle web
buildpact export-web

# Exportar com compressão máxima
buildpact export-web --compress high
```

Gera um bundle portátil que pode ser usado com interfaces de IA baseadas na web, como Claude.ai, ChatGPT ou Gemini.
