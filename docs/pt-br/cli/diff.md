# buildpact diff

Rastrear mudanças entre execuções do pipeline.

## Uso

```bash
buildpact diff
```

## Opções

Este comando não possui opções.

## Exemplos

```bash
# Mostrar mudanças desde a última execução do pipeline
buildpact diff
```

Mostra o que mudou desde a última execução de spec, plan ou execute, ajudando a entender o delta entre estágios do pipeline.
