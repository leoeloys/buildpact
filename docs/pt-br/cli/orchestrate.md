# buildpact orchestrate

Conheça o Pacto, seu orquestrador de projetos.

## Uso

```bash
buildpact orchestrate
```

## Opções

Este comando não possui opções.

## Exemplos

```bash
# Deixe o Pacto avaliar seu projeto e direcionar ao comando certo
buildpact orchestrate
```

O Pacto avalia o estado atual do projeto e automaticamente direciona você ao comando mais apropriado com base em onde você está no pipeline.
