# buildpact constitution

Criar ou editar regras do projeto.

## Uso

```bash
buildpact constitution
```

## Opções

Este comando não possui opções.

## Exemplos

```bash
# Abrir o editor interativo de constituição
buildpact constitution
```

Abre um editor interativo para gerenciar os padrões de código, regras de conformidade, restrições de arquitetura, portões de qualidade e regras específicas do domínio do seu projeto.
