# buildpact help

Mostrar comandos e status do projeto.

## Uso

```bash
buildpact help
```

## Opções

Este comando não possui opções.

## Exemplos

```bash
# Listar todos os comandos disponíveis
buildpact help
```

Exibe todos os comandos disponíveis com suas descrições e o status atual do projeto.
